map3d Engine Technical Manual

Description: This function manages 3D space for elements. It processes requests to position, rotate, scale, and skew objects by calculating a unified 16-number matrix. It updates elements immediately and ignores invalid or crashing data through built-in safety guards.

1. Input Targets

The input identifies the element to be moved and the specific settings for that movement.

  • The Target: An HTML object reference (e.g., coinFocus). Use null to just return the text string.
  • The Settings: An object containing movement labels (e.g., {z: 45}).

2. Movement Examples

4. Automatic Safety Guards

  • Memory: If settings are missing, the function reads the current values from the element automatically.
  • Scale Floor: Size is limited to 0.001 to prevent the object from vanishing or crashing.
  • Mirroring: Negative scale values (e.g., -1) are fully supported for mirroring and flipping.
  • Stability: Perspective and Skew are capped to prevent mathematical errors (division by zero).
