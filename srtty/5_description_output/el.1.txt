Αυτό το φροντιστήριο δίνει μια μέθοδο για τη χειροκίνητη μεταγραφή ενός βίντεο με σκοπό την επένδυση του αγωγού μεταφράσεων λεζάντα χρησιμοποιώντας LibreTranslate(argos translate). Η αποστολή των μεταφρασμένων αρχείων '. srt' στο 'YouTube' καλύπτεται επίσης. Αυτό είναι ένα σεμινάριο για τη δημιουργία των άλλων μαθημάτων μου. Το λογισμικό που χρησιμοποιείται είναι διαθέσιμο στο αποθετήριο GitHub για το έργο σύνταξης HTML. Τώρα ξέρω ότι έχω έρθει σε πλήρη κύκλο. Τώρα έχω κάνει ένα σεμινάριο για να κάνω μαθήματα.
