Ce tutoriel donne une méthode pour transcrire manuellement une vidéo dans le but d'aligner le pipeline de traduction de légende en utilisant LibreTrail(argos translate). Le téléchargement des fichiers '.srt' traduits vers 'YouTube' est également couvert. C'est un tutoriel sur la réalisation de mes autres tutoriels. Le logiciel utilisé est disponible au dépôt GitHub pour le projet de rédaction HTML. Maintenant je sais que j'ai fait plein cercle. J'ai maintenant fait un tutoriel sur la réalisation de tutoriels.
