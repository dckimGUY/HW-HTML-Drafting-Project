Deze tutorial geeft een methode voor het handmatig transcriberen van een video voor het opstellen van bijschrift vertaling pijplijn met behulp van LibreTranslate(argos vertalen). Het uploaden van de vertaalde '.srt' bestanden naar 'YouTube' wordt ook behandeld. Dit is een handleiding over het maken van mijn andere tutorials. De gebruikte software is beschikbaar in de GitHub repository voor het HTML Drafting Project. Nu weet ik dat ik de cirkel rond ben. Ik heb nu een tutorial gemaakt over het maken van tutorials.
