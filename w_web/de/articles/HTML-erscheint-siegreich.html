PCs, Mobile-Phones, Tablet-Computer: Sie alle benötigen Web-Browser, um die  nutzen zu können; internet !  HTML   ist eine sehr alte technologie, und die  internet   ist vollständig um sie herum. Es ist wie der Leim, der alles zusammenhält.

Browser sind nicht alle genau die gleichen, aber eine Sache, auf die sie alle zustimmen ist  HTML Es gibt sogar einen internationalen Standard dafür! Das nächste Mal scrollen Sie durch Ihre Lieblings-  webseite , denken sie einfach darüber nach, wie alles (fast) von dem, was sie sehen, ist nur regelmäßige alte  HTML Es ist schon lange her.

HTML ist ein Multimedia-Mastermind.

Das interessanteste an  HTML  ist, dass es wirkt wie ein fangnetz, das verwendet wird, um alle verschiedenen arbeit, die sie mit ihrem computer oder mobiltelefon tun können zusammen zu ziehen. Bilder, Video, Audio, Text, Animationen , und alle arten von interaktiven und programmierbaren sachen werden durch das wunder zusammengeführt, das ist unbsp; HTML Es ist wie eine Collage-Mischung, wo es keine Regeln gibt, und Sie können alles, was Sie wollen, in sie setzen.

HTML  ist die perfekte Sache, Projekte aller Art zu erstellen.

HW HTML-Entwurfprogramm kann für mehr als nur HTML verwendet werden...

Das stimmt, du werden lesen Sie das richtig. Dieses Programm kann ein kleines bildbearbeitung auch! Wie Sie sehen werden, ergibt das einen sehr guten Sinn. Wenn alle Ihre Bilder mit demselben Design-Gitter geschnitten werden, passt es immer wieder zusammen. So ist es wirklich schön, wenn man ein Bild in den Hintergrund ziehen kann ( alt + ziehen und fallen ) Dann ist es noch besser, wenn Sie dieses Bild mit dem  ausschneiden können; HTML   rechtecke ( verschiebung + doppelklick ). Was könnte das noch interessanter machen? Sie können auch die Bilder nach unten stempeln! ( alt + doppelklick )
Wie speichern Sie dann die Bilder zurück?: Sie können das Bild von jedem Rechteck speichern, indem Sie 'SHIFT + 1' drücken (auf den oberen Zeilennummern). So, dann können Sie an diesen kleinen Bildern in MS Paint arbeiten und dann ziehen Sie die Dateien direkt wieder in Platz.
