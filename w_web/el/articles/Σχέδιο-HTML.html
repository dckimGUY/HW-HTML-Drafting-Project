Τι είναι το HTML Σύνταξη?

Περίεργο που ρωτάς. HTML Σύνταξη, αν πάμε μπροστά και να ελέγξετε γύρω, είναι όταν αρχίζετε να κάνετε κάποια HTML και να χρησιμοποιήσετε μόνο το πιο βασικό επεξεργαστή κειμένου για να γράψετε όλα τα \\έντονα\\ γλώσσα σήμανση ρητά. Αυτό θα μάθετε αν ψάξετε το διαδίκτυο προσπαθώντας να καταλάβετε τι σημαίνει \\μοντάζ HTML\\. Μην πιστεύεις όλα όσα διαβάζεις!

Δεν θα έπρεπε η σύνταξη HTML να είναι κάτι οπτικό; Κάτι ευχάριστο; Κάτι διασκεδαστικό!

Ναι, φυσικά θα έπρεπε. Έτσι, δεν υπάρχει πραγματικά καμία ανάγκη να κατά τη διάρκεια όλο το χρόνο σας πληκτρολογώντας όλο τον κώδικα HTML, σε όλα είναι καταπληκτική λεπτομέρεια, απευθείας σε έναν επεξεργαστή κειμένου. Υπάρχουν κι άλλοι τρόποι. Αυτό που πραγματικά χρειάζεται να έχουμε είναι μια στρατηγική για τη σύνταξη εγγράφων HTML.

Η HTML είναι εξαιρετικά οπτική. Ό,τι γράφουμε στον επεξεργαστή κειμένου γίνεται οπτικό όταν φτάνει στο πρόγραμμα περιήγησης. Ο περιηγητής εξετάζει τον κώδικα HTML και στη συνέχεια αποφασίζει πώς πρέπει να εμφανίζεται. Οπότε, γιατί δεν προχωράμε και να σχεδιάσουμε το μεγαλύτερο μέρος του HTML μας απευθείας μέσα στο πρόγραμμα περιήγησης για αρχή; Δεν θα ήταν πιο εύκολο από το να μαντεύεις κάθε φορά?

Ναι, ακούγεται πιο εύκολο! Έτσι δεν είναι; Οπότε, ας προχωρήσουμε και να το κάνουμε πάντα έτσι.


Καλή απόφαση! Με αυτόν τον τρόπο μπορούμε απλά περιστασιακά να καταγράψουμε τις δικές μας ιστοσελίδες και όλα τα είδη των άλλων HTML-based πράγματα.
