Qu'est-ce qui rend HTML si spécial?

La chose la plus importante et la plus spéciale à propos de  HTML   est qu'il est universel, et chaque ordinateur l'utilise.   Presque 100% des ordinateurs utilisent  HTML  parce que presque 100% des ordinateurs ont des navigateurs web. PC, téléphones mobiles, ordinateurs tablettes : ils ont tous besoin de navigateurs Web pour pouvoir utiliser le   internet - ..; HTML   est une technologie très ancienne, et le  internet   est basé entièrement autour. C'est comme la colle qui colle tout ensemble.

Les navigateurs ne sont pas tous exactement les mêmes mais, une chose sur laquelle ils sont tous d'accord is  HTML . Il y a même une norme internationale pour elle! La prochaine fois que vous faites défiler votre favori  site web , il suffit de penser à la façon dont tout (presque) ce que vous voyez est juste ordinaire-vieil  HTML Ça fait longtemps qu'il est là.

HTML est un cerveau multimédia.

La chose la plus intéressante à propos de  HTML   est qu'il agit comme un filet d'attrape-tout qui est utilisé pour rassembler tous les travaux différents que vous pouvez faire avec votre ordinateur ou téléphone mobile. Images, Vidéo, Audio, Texte, Animations , et toutes sortes de choses interactives et programmables sont réunies par l'émerveillement qui est  HTML . C'est comme faire un mélange de collage où il n'y a pas de règles et vous pouvez mettre tout ce que vous voulez dedans.

C'est pourquoi il est tellement amusant de faire votre propre  HTML . C'est comme un dessin 'mélangé-média', comme en classe d'art ! La seule chose qui est différente est que nous travaillons déjà directement dans le format numérique dont nous avons besoin . si nous terminons notre travail en réunissant tout en un et un; HTML  document, puis nous pouvons le publier immédiatement sur Internet. HTML  est la chose parfaite pour élaborer des projets de toutes sortes.

HW programme de rédaction HTML peut être utilisé pour plus que juste HTML...

C'est vrai, toi sont lire cela correctement. Ce programme peut faire un petit montage d'image moi aussi ! Comme vous le verrez, c'est très logique. Si toutes vos images sont découpées à l'aide de la même grille de conception, alors elles se recoupent toujours. Donc, c'est vraiment sympa quand vous pouvez glisser une image dans le fond ( alt + glisser-déposer ). Alors c'est encore mieux quand vous pouvez couper cette image en utilisant le  HTML les rectangles de   ( changement + double clic ). Qu'est-ce qui pourrait rendre ça encore plus intéressant ? Vous pouvez également tamponner les images vers le bas! ( alt + double clic )
Alors comment sauvegarder les images?: Vous pouvez enregistrer l'image de n'importe quel rectangle en appuyant sur 'SHIFT + 1' (sur les numéros de ligne du haut). Donc, alors vous pouvez travailler sur ces petites images dans MS Paint et ensuite faire glisser les fichiers directement en place. C'est exactement ce que j'ai fait pour obtenir l'image de la signalisation DCKIM sur Yonge Street qui semble si bonne. J'ai également pu garder les fichiers plus petits en les séparant. Une image a en fait un niveau de détail plus élevé que l'autre.
