כל המאבק שעשיתי התברר לי כגורם טוב. כדי לספר את הסיפור בצורה נכונה: עבדתי כל כך קשה כדי ללמוד   HTML בסופו של דבר ויתרתי והתחלתי ללמוד   JavaScript תודה לאל (SWT) שיכולתי לעבור את הזמן המטריד הזה, ולבסוף בעזרת HIS, יצאתי מהצד השני. עיצבתי את התוכנית לאחר מה שבדרך כלל היית מצפה לקבל בכל תוכנית ציור גרפית רגילה. מה הופך את HTML לכל כך מיוחד?

הדבר החשוב והמיוחד ביותר ב-   HTML   הוא שזה אוניברסלי כל מחשב משתמש בו.     Almost 100% של מחשבים להשתמש   HTML כי כמעט 100% מהמחשבים יש דפדפנים. מחשבים, טלפונים ניידים, Tablet-Computers: כולם זקוקים לדפדפנים מקוונים כדי להיות מסוגלים להשתמש ב-   אינטרנט !   HTML   הוא טכנולוגיה עתיקה מאוד, ואת   אינטרנט   הוא מבוסס לחלוטין סביבו. זה כמו דבק שמדביק הכל ביחד.

דפדפנים הם לא בדיוק אותו הדבר, אלא דבר אחד שכולם מסכימים עליו הוא   HTML יש אפילו תקן בינלאומי לזה! בפעם הבאה אתה מגולל דרך האהובים עליך; אתר אינטרנט רק תחשוב על איך כל (כמעט) של מה שאתה רואה הוא רק בן רגיל; HTML זה באמת היה בסביבה במשך זמן רב.

HTML הוא מאסטרד מולטימדיה.

הדבר הכי מעניין ב-  HTML   הוא כי זה פועל כמו רשת לתפוס אשר משמש כדי למשוך יחד את כל העבודה אחרת שאתה יכול לעשות עם המחשב או הטלפון הנייד שלך. תמונות, וידאו, Audio, Text, Animations וכל סוגי הדברים האינטראקטיביים והתוכניתיים מובאים יחד דרך הפלא שהוא   HTML זה כמו לעשות תערובת קולגן שבו אין כללים ואתה יכול לשים כל דבר שאתה רוצה בו.

לכן זה כל כך כיף לעשות את עצמך; HTML . זה הרבה כמו ציור 'מתערבב', בדיוק כמו בכיתה לאמנות! הדבר היחיד שונה זה אנחנו כבר עובדים ישירות בפורמט הדיגיטלי שאנחנו צריכים אם נסיים את עבודתנו על ידי הבאת הכל ביחד ל-  HTML ואחרי הכל, אנחנו יכולים לפרסם אותו מיד באינטרנט. HTML   הוא הדבר המושלם לייסד פרויקטים מכל הסוגים.

תכנית HW HTML Drafting ניתן להשתמש עבור יותר מ-HTML.

זה נכון, אתה הם לקרוא את זה נכון. תוכנית זו יכולה לעשות תמונה קטנה גם! כפי שתראו, זה הגיוני מאוד. אם כל הדימויים שלך נחתכים באמצעות אותה רשת עיצוב, אז זה תמיד מתאים בחזרה יחד. אז זה באמת נחמד כאשר אתה יכול לגרור תמונה לתוך הרקע ( alt + גרור וטיפה ). אז זה אפילו יותר טוב כאשר אתה יכול לחתוך את התמונה הזו באמצעות ה-   HTML אנגלית:rectangles שינוי + לחיצה כפולה ). מה יכול לעשות את זה אפילו יותר מעניין? אתה יכול גם להרים את התמונות בחזרה למטה alt + לחיצה כפולה )
