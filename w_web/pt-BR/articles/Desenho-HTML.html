O que é HTML Redação?

Engraçado você perguntar. HTML Redação, se formos em frente e verificarmos, é quando você começa a inventar HTML e usa apenas o editor de texto mais básico para escrever toda a linguagem de marcação 'intensa' explicitamente. Isso é o que você vai descobrir se você procurar na web tentando entender o que 'HTML Redação' significa. Não acredite em tudo que lê!

Não deveria ser algo visual? Algo agradável? Algo divertido!

Sim, claro que deveria. Então, não há necessidade de você digitar todo o código HTML, em todos os detalhes, diretamente em um editor de texto. Há outras maneiras de fazer isso. O que realmente precisamos ter é uma estratégia para Redação de documentos HTML.

HTML é extremamente visual. Tudo que escrevemos no editor de texto se torna algo visual quando chega ao navegador. O navegador olha o código HTML e decide como deve ser exibido. Então, por que não vamos em frente e elaboramos a maioria do nosso HTML diretamente dentro do navegador para começar? Não seria muito mais fácil do que adivinhar cada vez?

Sim, isso parece mais fácil! Não é? Então, vamos em frente e sempre fazemos assim.


Boa decisão! Assim podemos escrever casualmente nossos próprios sites e todo tipo de outras coisas baseadas em HTML.
