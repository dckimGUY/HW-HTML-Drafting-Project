Ce face HTML atât de special?

Cel mai important și special lucru despre   HTML   este că este universal, și fiecare computer îl utilizează.    Aproape 100% din utilizarea calculatoarelor  HTML   deoarece aproape 100% din calculatoare au browsere web. PC-uri, telefoane mobile, tabletă-computer: Toate au nevoie de browsere web pentru a putea utiliza   internet   HTML  este o tehnologie foarte veche, și  internet   este bazat complet în jurul ei. E ca lipiciul care leagă totul.

Browser-urile nu sunt toate exact la fel, dar, un lucru pe care toți sunt de acord cu privire la este  HTML Există chiar și un standard internațional pentru ea! Data viitoare când sunt defilare prin favorite  website , gândiți-vă doar la modul în care toate (aproape) ceea ce vedeți este doar regulat-vechi  HTML De fapt e pe aici de mult timp.

HTML este un geniu multimedia.

Cel mai interesant lucru despre  HTML  este ca actioneaza ca o plasa care este folosita pentru a aduna toate lucrarile diferite pe care le puteti face cu computerul sau telefonul mobil. Imagini, Video, Audio, Text, Animaţii , și tot felul de lucruri interactive și programabile este adus împreună prin minunea care este  HTML E ca şi cum ai face un amestec de colaj unde nu există reguli şi poţi pune orice vrei în el.

De aceea este atât de distractiv să-ţi faci propria afacere; HTML . E ca un desen mixed-media, la fel ca la ora de artă! Singurul lucru care este diferit este că deja lucrăm direct în formatul digital de care avem nevoie . Dacă vom termina munca noastră prin aducerea totul împreună în an  HTML   document, apoi îl putem publica imediat pe internet. HTML este lucrul perfect pentru a elabora proiecte de toate tipurile.

Programul de redactare HW HTML poate fi folosit pentru mai mult de HTML...

Aşa e, tu sunt am citit corect. Acest program poate face unele mici timp de editare imagine şi eu! După cum veţi vedea, are sens. Dacă toate imaginile sunt tăiate folosind aceeaşi reţea de design, atunci întotdeauna se potriveşte la loc. Deci, este foarte frumos atunci când puteți trage o imagine în fundal ( alt + drag and drop ).
