การร่างแบบ HTML คืออะไร?

ตลกดีที่ถาม HTML แบบร่าง, ถ้าเราไปข้างหน้าและตรวจดูรอบๆ, คือเมื่อคุณเริ่มสร้าง HTML ขึ้นมา และใช้เฉพาะเครื่องมือแก้ไขข้อความพื้นฐานที่สุด นั่นคือสิ่งที่คุณจะพบ ถ้าคุณค้นหาเว็บ พยายามทําความเข้าใจว่า 'HTML simpleing' หมายถึงอะไร อย่าเชื่อทุกอย่างที่อ่าน!

การร่างแบบ HTML ควรเป็นสิ่งที่เห็นไม่ใช่เหรอ มีอะไรสนุกๆเหรอ? บางอย่างที่สนุก!

ใช่ แน่นอน มันควรเป็นอย่างนั้น ดังนั้น จึงไม่จําเป็นจริงๆ ในขณะที่เวลาทั้งหมดของคุณตลอดเวลา พิมพ์รหัส HTML ทั้งหมด ในรายละเอียดที่น่าตื่นตาตื่นใจทั้งหมด มีวิธีอื่นที่จะทํา สิ่งที่เราต้องการจริงๆ คือกลยุทธ์ในการร่างเอกสาร HTML.

ภาพ HTML เป็นภาพที่ยอดเยี่ยม ทุกอย่างที่เราเขียนลงในเครื่องมือแก้ไขข้อความ กลายเป็นภาพบางอย่าง เมื่อมันเข้าถึงเบราว์เซอร์ เบราว์เซอร์ดูที่รหัส HTML แล้วตัดสินใจว่าจะให้แสดงอย่างไร ดังนั้น ทําไมเราไม่ลองไปวาด HTML ส่วนใหญ่ของเรา โดยตรงภายในเว็บเบราว์เซอร์เพื่อเริ่มต้น? มันจะไม่ง่ายกว่าการเดาทุกครั้งเหรอ?
